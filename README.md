# NOMARU Essential Oils Website

A modern, minimal marketing website for NOMARU, a South African natural essential oil manufacturer.

## Features
- **Glassmorphism Design**: Modern fintech-inspired UI with frosted glass effects.
- **Responsive**: Mobile-first design that works on all devices.
- **Animations**: Smooth scroll-reveal and hover effects.
- **Static Architecture**: Built with pure HTML5, CSS3, and Vanilla JavaScript.

## Tech Stack
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- Vanilla JavaScript
- Lucide Icons (via CDN)
- Google Fonts (Inter)

## Running Locally
1. Clone the repository or download the files.
2. Open `index.html` in any modern web browser.
3. No build step or local server is required for basic viewing, though a local server (like Live Server) is recommended for the best experience.

## Deploying to GitHub Pages
1. Create a new repository on GitHub.
2. Push all project files to the `main` branch.
3. Go to the **Settings** tab of your repository.
4. Navigate to **Pages** in the left sidebar.
5. Under **Build and deployment**, ensure "Deploy from a branch" is selected.
6. Select the `main` branch and the `/ (root)` folder.
7. Click **Save**.
8. Your site will be live at `https://<your-username>.github.io/<repository-name>/` within a few minutes.
